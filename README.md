 Weather Web App

A simple weather application built using HTML, CSS, and JavaScript that fetches real-time weather data using an API.



 Features
- Search weather by city name
- Displays temperature, weather condition, and humidity
- Dynamic UI updates
- Simple and clean design



 Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeatherMap API



How to Run the Project

1. Download or clone this repository
2. Open `script.js`
3. Replace:
   ```js
   let apiKey = "YOUR_API_KEY"; with the new api key you can get from this website https://openweathermap.org/api and save it 
4. Open html file and type your city name and search for weather details.
